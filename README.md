# paq

paq files to hash.

Hash a single file or all files in directory recursively.

## Installation

1. Clone repository.
2. Run `cargo install --path .` from root of repository.

## Usage

Run `paq [src]` to hash src (file or directory). 

For help, run `paq --help`.
